¿Que es JavaScript?

JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web interactivas. Es uno de los tres lenguajes fundamentales para el desarrollo web junto con HTML (estructura) y CSS (estilos). JavaScript permite agregar funcionalidades dinámicas y comportamientos a una página web, como:

Interactividad con el usuario: Capturar eventos como clics, desplazamientos o introducción de datos en formularios y responder a ellos.
Manipulación del DOM (Document Object Model): JavaScript puede cambiar el contenido, la estructura y el estilo de una página web en tiempo real, por ejemplo, mostrando o ocultando elementos, modificando el contenido de una etiqueta, etc.
Validación de formularios: Comprobar si los datos ingresados en un formulario son correctos antes de enviarlos al servidor.
Actualizaciones dinámicas: Con JavaScript y tecnologías como AJAX, es posible cargar o actualizar datos en una página web sin necesidad de recargarla.
Animaciones: Crear animaciones, como desplazamientos suaves, efectos visuales y transiciones. Juegos y gráficos: Crear juegos y gráficos interactivos utilizando bibliotecas como Canvas o WebGL. Características clave: Interpretado: JavaScript es un lenguaje interpretado, lo que significa que el navegador lo ejecuta directamente sin necesidad de compilarlo previamente. Basado en eventos: Muchas acciones en JavaScript dependen de eventos del usuario, como clics o teclas presionadas. Lenguaje del lado del cliente y del servidor: Aunque tradicionalmente ha sido usado en el lado del cliente (navegador), también puede ejecutarse en el servidor con plataformas como Node.js.

Ejercicios con ciclos

Propósito de los bucles:
Repetir acciones: Ejecutar un bloque de código un número específico de veces.
Recorrer estructuras de datos: Iterar sobre elementos de arrays, objetos o listas para acceder y manipular sus valores.
Automatizar procesos: Realizar operaciones repetitivas sin tener que escribir manualmente el mismo código múltiples veces.


Tipo de Bucle Descripción Uso Común
for Ideal cuando se conoce la cantidad exacta de veces que se debe ejecutar el código. Iterar sobre un array con un número definido de elementos.
while Ejecuta un bloque de código mientras una condición específica se mantenga verdadera. Cuando no se sabe cuántas veces se ejecutará el código, como esperar una condición externa.
do...while Ejecuta un bloque de código al menos una vez y luego repite el bloque mientras la condición especificada sea verdadera. Cuando se necesita asegurarse de que el bloque de código se ejecute al menos una vez, como en la validación de entradas.
forEach Método utilizado para iterar sobre los elementos de un array. Iterar sobre un array y realizar una operación en cada elemento.
for...of Método utilizado para iterar sobre los elementos de un iterable (como un array o una cadena). Cuando se desea recorrer los valores de un iterable sin preocuparse por los índices.

Librerias JavaScript CDN

Las bibliotecas CDN (Content Delivery Network) en JavaScript son versiones alojadas en la nube de bibliotecas y archivos JavaScript populares (como jQuery, React, Bootstrap, etc.) que puedes integrar en tus proyectos mediante enlaces directos. Al incluir una biblioteca desde un CDN en lugar de alojarla localmente en tu servidor o proyecto, obtienes beneficios como: Mejor rendimiento y carga más rápida: Los CDN están distribuidos en múltiples ubicaciones alrededor del mundo. Cuando un usuario accede a tu sitio, se carga la biblioteca desde el servidor más cercano, reduciendo el tiempo de carga. Aprovechar el caché del navegador: Como muchos sitios usan las mismas versiones de bibliotecas populares (por ejemplo, jQuery), los navegadores suelen tener esos archivos ya en caché. Si un usuario ya cargó la biblioteca de un CDN en otro sitio, se evitará volver a descargarla, mejorando el rendimiento. Reducción del ancho de banda en tu servidor: Al cargar las bibliotecas desde un CDN, reduces la carga en tu propio servidor, ya que estos archivos se cargan desde el CDN en lugar de consumir ancho de banda en tu servidor. Actualización fácil: Las bibliotecas en los CDN suelen actualizarse rápidamente. Los desarrolladores pueden acceder a nuevas versiones cambiando solo el enlace.

Ejemplo de uso de React con CDN Ejemplo complejo utilizando React.createElement (sin JSX): Este ejemplo incluye un formulario para agregar elementos a una lista, con un estado que mantiene la lista y permite agregar elementos dinámicamente.
CODIGO DEL EJEMPLO
<div id="root"></div>
<script type="text/javascript">
// Crear un componente funcional de React
const App = () => {
// Estado para la lista de elementos
const [items, setItems] = React.useState([]);
// Estado para el texto del formulario
const [newItem, setNewItem] = React.useState('');
// Función para agregar un nuevo elemento a la lista
const addItem = () => {
if (newItem) {
setItems([...items, newItem]);
setNewItem(''); // Limpiar el campo de entrada
}
};
// Función para manejar el cambio en el input
const handleInputChange = (event) => {
setNewItem(event.target.value);
};
// Crear los elementos visuales usando React.createElement
return React.createElement('div', null,
React.createElement('h1', null, 'Lista de Tareas'),
React.createElement('input', {
type: 'text',
value: newItem,
onChange: handleInputChange,
placeholder: 'Escribe una tarea...'
}),
React.createElement('button', { onClick: addItem }, 'Agregar tarea'),
// Mostrar la lista de tareas
React.createElement('ul', null,
items.map((item, index) =>
React.createElement('li', { key: index }, item)
)
)
);
};

// Renderizar el componente en el div con id "root"
ReactDOM.render(React.createElement(App), document.getElementById('root'));
</script>

LIBRERIA CHART.JS
¿Qué es Chart.js? Chart.js es una biblioteca de JavaScript simple y flexible que permite crear gráficos interactivos en aplicaciones web. Utiliza un elemento HTML canvas para renderizar los gráficos y es ampliamente utilizada debido a su facilidad de uso, diseño atractivo y compatibilidad con múltiples tipos de gráficos, como gráficos de líneas, barras, radar, pastel, burbuja y más.
Instalación Primero, necesitas agregar Chart.js a tu proyecto. Esto se puede hacer mediante un CDN o instalándolo con un gestor de paquetes como npm.
SI QUIERES MAS INFORMACION SOBRE ESTA LIBRERIA TE DEJARE EL LINK DE SU DOCUMENTACION PARA QUE LA ESTUDIES MAS A FONDO : documentacion

Componente Descripción Ejemplo
type Especifica el tipo de gráfico que deseas crear. 'bar', 'line', 'pie', 'doughnut', 'radar', 'polarArea', 'bubble', 'scatter'.
data Contiene los datos del gráfico, incluidos las etiquetas y conjuntos de datos. { labels: ['A', 'B'], datasets: [{ data: [10, 20] }] }
labels Etiquetas del eje X (o categorías en gráficos circulares). ['Enero', 'Febrero', 'Marzo']
datasets Configura los datos y el estilo para los gráficos. { label: 'Ventas', data: [30, 50, 70], backgroundColor: ['red', 'blue', 'green'] }
options Configura el comportamiento y la apariencia general del gráfico. { scales: { y: { beginAtZero: true } } }
backgroundColor Define el color de fondo de los elementos del gráfico. ['rgba(255, 99, 132, 0.2)', 'rgba(54, 162, 235, 0.2)']
borderColor Color del borde de los elementos. ['rgba(255, 99, 132, 1)', 'rgba(54, 162, 235, 1)']
borderWidth Grosor del borde de los elementos. 1
scales Configura los ejes del gráfico (aplicable en ciertos tipos de gráficos). { y: { beginAtZero: true } }
responsive Determina si el gráfico debe ajustarse automáticamente al tamaño del contenedor. true (por defecto).

LIBRERIA LUXON
Luxon es una librería de JavaScript para el manejo de fechas y horas. Fue creada por uno de los desarrolladores de Moment.js, y es una alternativa más moderna y eficiente para trabajar con fechas, horas, zonas horarias y formatos. Luxon se basa en el estándar Intl de JavaScript, lo que le da un rendimiento más rápido y una integración más fluida con las APIs nativas del navegador.
Algunas de las características clave de Luxon son:
Soporte completo de zonas horarias a través de la biblioteca IANA Time Zones.
Formato de fechas y horas flexible, soportando varios formatos predeterminados o personalizados.
Manipulación de fechas mediante operaciones como sumar o restar días, meses, horas, etc.
Localización para trabajar con fechas en diferentes idiomas y configuraciones regionales.
API fácil de usar que permite trabajar con objetos de fecha y hora de manera clara y concisa.


Categoría Función Descripción
Creación de objetos DateTime
Creación de fecha y hora actual `DateTime.now()` Obtiene la fecha y hora actual.
Crear fecha con la hora local `DateTime.local()` Crea un objeto DateTime con la fecha y hora local.
Crear fecha con la hora UTC `DateTime.utc()` Crea un objeto DateTime con la fecha y hora en formato UTC.
Crear a partir de una cadena ISO `DateTime.fromISO()` Crea un objeto DateTime a partir de una cadena ISO 8601.
Crear a partir de milisegundos `DateTime.fromMillis()` Crea un objeto DateTime a partir de milisegundos.
Crear a partir de un formato personalizado `DateTime.fromFormat()` Crea un objeto DateTime a partir de una cadena con un formato específico.
Obtención de componentes
Obtener el año `DateTime.year` Obtiene el año de la fecha.
Obtener el mes `DateTime.month` Obtiene el mes de la fecha (1-12).
Obtener el día `DateTime.day` Obtiene el día del mes.
Obtener el día de la semana `DateTime.weekday` Obtiene el día de la semana (1-7, donde 1 es lunes).
Obtener la hora `DateTime.hour` Obtiene la hora en formato de 24 horas.
Obtener los minutos `DateTime.minute` Obtiene los minutos de la hora.
Obtener los segundos `DateTime.second` Obtiene los segundos.
Obtener los milisegundos `DateTime.millisecond` Obtiene los milisegundos.
Formateo y salida
Convertir a formato personalizado `DateTime.toFormat()` Convierte la fecha a un formato personalizado.
Convertir a formato ISO `DateTime.toISO()` Convierte la fecha a formato ISO 8601.
Convertir a formato local `DateTime.toLocaleString()` Convierte la fecha a una cadena legible en la configuración regional.
Convertir a formato HTTP `DateTime.toHTTP()` Convierte la fecha a formato HTTP (RFC 2822).
Manipulación de fechas
Sumar tiempo a una fecha `DateTime.plus()` Suma una cantidad de tiempo a la fecha (por ejemplo, días, horas, meses).
Restar tiempo a una fecha `DateTime.minus()` Resta una cantidad de tiempo a la fecha (por ejemplo, días, horas, meses).
Modificar un componente de la fecha `DateTime.set()` Modifica un componente específico de la fecha (por ejemplo, año, mes, hora).
Establecer al inicio de una unidad `DateTime.startOf()` Establece la fecha al inicio de la unidad de tiempo (por ejemplo, primer día del mes).
Establecer al final de una unidad `DateTime.endOf()` Establece la fecha al final de la unidad de tiempo (por ejemplo, último día del mes).
Comparaciones
Comparar dos fechas `DateTime.equals()` Compara dos objetos DateTime para ver si son iguales.
Comprobar si es anterior a otra fecha `DateTime.isBefore()` Verifica si una fecha es anterior a otra.
Comprobar si es posterior a otra fecha `DateTime.isAfter()` Verifica si una fecha es posterior a otra.
Comprobar si las fechas son iguales en una unidad `DateTime.hasSame()` Verifica si dos fechas tienen el mismo componente (día, mes, año, etc.).
Zonas horarias
Establecer zona horaria `DateTime.setZone()` Establece la zona horaria de la fecha.
Obtener el nombre de la zona horaria `DateTime.zoneName` Obtiene el nombre de la zona horaria (por ejemplo, "America/New_York").
Obtener el desplazamiento en minutos `DateTime.offset` Obtiene el desplazamiento de la zona horaria en minutos (por ejemplo, -300 para GMT-5).
Válido y no válido
Verificar si es válido `DateTime.isValid` Verifica si un objeto DateTime es válido.
Crear objeto DateTime inválido `DateTime.invalid` Marca un objeto DateTime como inválido.
